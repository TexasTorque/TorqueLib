Texas Torque Autonomous Framework
=================================

A collections of Texas Torque autonomous
abstract classes and boilerplate. Created
for use in the 2022 Rapid React season.
Included in the 2023 TorqueLib and project
refactoring.

The framework provides the following classes:
- TorqueAssist (currently not included)
- TorqueAutoManager
- TorqueBlock
- TorqueCommand
- TorqueEmpty
- TorqueSequence
